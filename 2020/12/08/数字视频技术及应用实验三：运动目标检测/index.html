# 数字视频技术及应用实验三：运动目标检测

# 实验内容及要求

阅读实验三指导书，任选一种方法（背景减法、帧差法等）编程实现视频中运动目标检测，并分析实验结果。

# 实验内容指导

1. 在 vs 中新建工程；
2. 配置 OpenCV；
3. 定义读取视频的变量
4. 定义相应的指针和结构并初始化
5. 将视频灰度化，并将指针的内容存到相应的结构体中
6. 计算视频中前后两帧的差值，并将其二值化；
7. 更新背景，并在二值图像中检索轮廓
8. 在原视频中框出运动物体
9. 在视频中显示最终结果
