# Hexo 搭建个人博客（一）

# 前期准备（软件环境配置）

# 1、Nodejs

下载链接：http://nodejs.cn/download/

配置：

  • 安装教程：https://www.runoob.com/nodejs/nodejs-install-setup.html
  • 检验安装配置成功方法：快捷键 win + r 输入 cmd ，打开命令提示符，输入 node -v 和 npm -v 能正常显示版本号即为成功。

# 2、Git

下载链接：https://git-scm.com/download/

配置：

  • 安装教程：https://www.runoob.com/git/git-install-setup.html
  • 检验安装配置成功方法：快捷键 win + r 输入 cmd ，打开命令提示符，输入 git --version 能正常显示版本号即为成功。（鼠标右键会有两个新的右键菜单 Git GUI Here 和 Git Bash Here )

# 3、Typora（可选）(Markdown 编辑器)

下载链接：https://typora.io/#download

配置：

  • 直接下载安装即可
  • (在线 Markdown 编辑器：https://pandao.github.io/editor.md/)
